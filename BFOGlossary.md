| Adequatism.                                      |                   | [Adequatism] | 提案 | The goal of ontology as viewed by the proponent of adequatism is to do justice to the vast array <br> of different kinds of entities that exist in the world, rather than ignoring these or those  <br>specific kinds of entities or attempting to explain them away. | The view that the entities in any given domain should be taken seriously on their own terms (as contrasted with reductionism). <br>The goal of adequatism is to do justice to all the different kinds of entities that exist in reality; see also General principles  of ontology design.                                                                                                                                                                                                                                                                  |
| All-some structure.                              |                   |              |      |                                                                                                                                                                                                                                                                       | The structure that applies to the relations obtaining between universals whereby if a universal C bears some relation R to a <br> universal D, then all relevant instances of C must bear the relevant instance-level relation to some instance of D at all relevant  times.                                                                                                                                                                                                                                                                               |
| Aristotelian definitional structure.             |                   |              |      |                                                                                                                                                                                                                                                                       | A definition that has the basic form “An A is a B that Cs” where “A” is the definiendum, the term that is being defined; “B that Cs”<br> is the definiens, the expression that does the defining; and “C” is a statement of the differentia, that is to say, a statement of <br> what it is that marks out those instances of B that are As (those features  Bs must possess in order to be As).                                                                                                                                                           |
| Basic Formal Ontology (BFO).                     | 基本形式本体(BFO) |              |      |                                                                                                                                                                                                                                                                       | A top-level (or upper-level) ontology consisting of continuants and occurrents developed to support integration especially of data <br> obtained through scientific research. BFO is deliberately designed to be very small, in order to represent in consistent <br>fashion those top-level categories common to domain ontologies developed by scientists in different fields. BFO assists <br>domain ontologists by providing  a common top-level structure to support the interoperability of the multiple domain ontologies <br>created in its terms. |
| Category.                                        | 范畴              |              |      |                                                                                                                                                                                                                                                                       | A formal (which is to say, domain-neutral) universal, such as entity, continuant, occurrent.                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| Class.                                           | 类                |              |      |                                                                                                                                                                                                                                                                       | A maximal collection of particulars falling under a given general term; also called the extension of the term (or of the universal that the term denotes).                                                                                                                                                                                                                                                                                                                                                                                                 |
| Continuant.                                      | 持续体            |              |      |                                                                                                                                                                                                                                                                       | An entity that continues or persists through time, including (1) independent objects, (2) qualities and dispositions, and (3) the <br>spatial regions these entities occupy at any given time. Continuant and occurrent are the two highest categories (universals) in BFO.                                                                                                                                                                                                                                                                                |
| Continuant fiat  boundary.                       |                   |              |      |                                                                                                                                                                                                                                                                       | An immaterial entity that is of zero, one, or two dimensions and does not include a spatial region as part. Intuitively, a continuant<br> fiat boundary is a boundary of some material entity that exists exactly where that object meets its surroundings.                                                                                                                                                                                                                                                                                                |
| Controlled vocabulary.                           | 受控词表          |              |      |                                                                                                                                                                                                                                                                       | A collection of preferred terms that are used to promote consistent description and retrieval of data.                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| Defined  class.                                  |                   |              |      |                                                                                                                                                                                                                                                                       | A collection of individuals  that are grouped together by virtue of their exhibiting some combination of characteristics that does not <br>correspond to any universal.                                                                                                                                                                                                                                                                                                                                                                                    |
| Description logic (DL).                          | 描述逻辑          |              |      |                                                                                                                                                                                                                                                                       | A fragment  of first-order logic  (FOL) used for purposes of formal knowledge representation and having more efficient decision<br> properties than FOL. See also Web Ontology Language (OWL).                                                                                                                                                                                                                                                                                                                                                             |
| Disposition.                                     |                   |              |      |                                                                                                                                                                                                                                                                       | A realizable entity (a power, potential, or tendency) that exists because of certain features of the physical makeup of the <br>independent continuant that is its bearer.                                                                                                                                                                                                                                                                                                                                                                                 |
| Domain.                                          |                   |              |      |                                                                                                                                                                                                                                                                       | A delineated portion or sphere of reality corresponding to a scientific discipline (such as cell biology or electron microscopy), <br>or  to  an area of  knowledge  or  interest such as the  Great War or  stamp collecting.                                                                                                                                                                                                                                                                                                                             |
| Domain  ontology.                                | 领域本体           |              |      |                                                                                                                                                                                                                                                                       | See Ontology, domain.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| Entity.                                          |                   |              |      |                                                                                                                                                                                                                                                                       | Anything that exists.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| Epistemological idealism.                        |                   |              |      |                                                                                                                                                                                                                                                                       | See Idealism.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| Fallibilism.                                     |                   |              |      |                                                                                                                                                                                                                                                                       | The  view  that,  although  our  current  scientific  theories  are  the  best  candidates we  have  for representing the  truth  about<br>  reality,  it  may  nevertheless be  the  case at  any  given  stage that elements of our best current knowledge are <br>incorrect; see General principles  of ontology design.                                                                                                                                                                                                                                |
| Fiat object  part.                               |                   |              |      |                                                                                                                                                                                                                                                                       | A material entity that is a proper part of some larger object, but is not demarcated from the remainder of this object by any <br>physical discontinuities (thus, it is not itself an object). Examples include your upper torso, the Western hemisphere of the <br>planet earth.                                                                                                                                                                                                                                                                          |
| First-order  logic (FOL).                        |                   |              |      |                                                                                                                                                                                                                                                                       | A formal language and system of reasoning utilizing predicates, constants and variables, quantifiers (all, some, none), and logical<br> connectives (and, not, or, material implication);  also known as first-order predicate logic; see also Description logic (DL).                                                                                                                                                                                                                                                                                     |
| Foundational relations.                          |                   |              |      |                                                                                                                                                                                                                                                                       | The fundamental relations in BFO, especially is_a (meaning “is a subtype of”) and relations of part to whole.                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| Function.                                        |                   |              |      |                                                                                                                                                                                                                                                                       | A realizable entity whose realization is an end- or goal-directed activity of its bearer that exists in the bearer in virtue of its having<br> a certain physical makeup as a result of either natural selection (in the case of biological entities) or intentional design<br> (in the case of artifacts).                                                                                                                                                                                                                                                |
| General principles  of ontology design.          | 本体设计一般原则  |              |      |                                                                                                                                                                                                                                                                       | The principles to be applied in every process of designing an ontology, including realism, perspectivalism, adequatism, fallibilism,<br> the open-world assumption, and the low-hanging fruit  principle.                                                                                                                                                                                                                                                                                                                                                  |
| Generically dependent continuant.                |                   |              |      |                                                                                                                                                                                                                                                                       | A continuant that is dependent on one or other independent continuants and can migrate from one bearer to another through<br> a process of copying. We can think of generically dependent continuants as complex continuant patterns either of the sort <br>created by authors or designers or (in the case of DNA sequences) brought into being through the processes of evolution.                                                                                                                                                                       |
| Granularity.                                     |                   |              |      |                                                                                                                                                                                                                                                                       | The distinction between levels of reality exemplified for example in the biological domain in the levels of cells, organs, organisms and populations. See Perspectivalism.                                                                                                                                                                                                                                                                                                                                                                                 |
| History.                                         |                   |              |      |                                                                                                                                                                                                                                                                       | A BFO: process that is the sum of the totality of processes taking place in the spatiotemporal region occupied by a material entity or site.                                                                                                                                                                                                                                                                                                                                                                                                               |
| Idealism.                                        |                   |              |      |                                                                                                                                                                                                                                                                       | The thesis that our perceptions, thoughts, and statements are not about reality, but are rather about certain mental or created <br>objects (variously called appearances, concepts, ideas, or models). For the idealist, there is nothing that exists (or, for the <br>defender of Epistemological idealism, nothing that can be known) outside of the realm of sensations or ideas or concepts; <br>see also Realism.                                                                                                                                    |
| Immaterial entity.                               |                   |              |      |                                                                                                                                                                                                                                                                       | An independent continuant that contains no material entities as parts. Immaterial entities divide into two major subgroups: (1) <br>boundaries and sites, which bound, or are demarcated in relation to, material entities, and which can thus change location,<br> shape, and size as their material hosts move or change shape or size; (2) spatial regions, which exist independently<br> of material entities, and which thus do not change.                                                                                                           |
| Independent continuant.                          | 独立持续体        |              |      |                                                                                                                                                                                                                                                                       | A continuant entity that is the bearer of qualities and a participant in processes. Independent continuants are such that their <br>identity can be maintained over time through gain and loss of parts, as well as through changes in qualities.                                                                                                                                                                                                                                                                                                          |
| Individual.                                      | 个体              |              |      |                                                                                                                                                                                                                                                                       | See Particular.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| Inherence.                                       |                   |              |      |                                                                                                                                                                                                                                                                       | A one-sided dependence that obtains between specifically and generically  dependent continuants on the one hand and <br>independent continuants on the other. Qualities,  dispositions,  and roles inhere in independent continuants.                                                                                                                                                                                                                                                                                                                      |
| Instance.                                        | 实例              |              |      |                                                                                                                                                                                                                                                                       | See Particular.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| Instantiation.                                   |                   |              |      |                                                                                                                                                                                                                                                                       | A relation between a particular and a universal; the particular is one of an open-ended set of particulars that are similar in the <br>relevant respect; the particular is such that if it ceases to be an instance of this universal then it ceases to exist.                                                                                                                                                                                                                                                                                             |
| Inventory.                                       |                   |              |      |                                                                                                                                                                                                                                                                       | A representational artifact consisting of entries designed to keep track of the particulars contained, for example, in a warehouse <br>of products.                                                                                                                                                                                                                                                                                                                                                                                                        |
| is_a.                                            |                   |              |      |                                                                                                                                                                                                                                                                       | The subtype relation used to form the backbone taxonomic hierarchy of an ontology.                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| Low-hanging fruit  principle.                    | 低果先摘原则      |              |      |                                                                                                                                                                                                                                                                       | The principle that states that the designer of a domain ontology should start with those (often trivial) features of the subject <br>matter that are the easiest and clearest to understand; also see General principles of ontology design.                                                                                                                                                                                                                                                                                                               |
| Material  entity.                                |                   |              |      |                                                                                                                                                                                                                                                                       | An  independent  continuant  that  has some  portion   of  matter  as part,  is  spatially  extended in  three  dimensions,  and  that<br>   continues  to   exist  through   some  interval  of  time,   however short. Three principal subtypes of material entity are<br> object, fiat object part, and object aggregate.                                                                                                                                                                                                                               |
| Nominalism.                                      |                   |              |      |                                                                                                                                                                                                                                                                       | The  position   that   there   are  no   universals,   in   contrast   to   ontological  realism;   also  see Representationalism.                                                                                                                                                                                                                                                                                                                                                                                                                         |
| Object.                                          |                   |              |      |                                                                                                                                                                                                                                                                       | A material   entity that  is (1)  spatially extended  in  three  dimensions; (2)  causally unified;  and (3)  maximally  self-connected. Examples include a single cell, a laptop,  an organism, a planet, a spaceship.                                                                                                                                                                                                                                                                                                                                    |
| Object aggregate.                                |                   |              |      |                                                                                                                                                                                                                                                                       | A material  entity that has as parts (exactly) two or more objects that are separate from each other in the sense that they share <br>no parts in common. Examples include a heap of stones, a population  of bacteria, a flock of geese.                                                                                                                                                                                                                                                                                                                  |
| Occurrent.                                       | 过程体            |              |      |                                                                                                                                                                                                                                                                       | An entity that unfolds itself in time; the BFO category of occurrents comprises not only (1) the processes that  unfold  themselves<br> in their successive temporal phases, but  also (2) the boundaries or thresholds at the beginnings or ends of<br> such temporal phases, as well as (3) the temporal and spatio- temporal regions  in which these processes occur. <br>Occurrent  and continuant are the two highest categories (universals) in BFO.                                                                                                 |
| One-dimensional continuant fiat  boundary.       |                   |              |      |                                                                                                                                                                                                                                                                       | A  continuant  fiat   boundary    that   is  a  continuous   fiat   line  whose  location   is  defined   in relation  to  some material   <br>entity. Examples include  the  boundary  of  a real estate parcel, your waistline.                                                                                                                                                                                                                                                                                                                          |
| One-dimensional spatial region.                  | 一维空间区域      |              |      |                                                                                                                                                                                                                                                                       | A spatial  region  with  one dimension, also called a spatial line, defined relative to some reference frame; for example, lines of <br>latitude and longitude.                                                                                                                                                                                                                                                                                                                                                                                            |
| One-dimensional temporal region.                 | 一维时间区域      |              |      |                                                                                                                                                                                                                                                                       | A  temporal  region   that   is  extended   in   time.   It   has  further   temporal   regions  as  parts. One-dimensional  temporal  <br> regions  are  the   temporal   regions  in   which   processes occur  or unfold.                                                                                                                                                                                                                                                                                                                               |
| Ontological realism.                             |                   |              |      |                                                                                                                                                                                                                                                                       | The  view  according   to   which   the   general  truths   about   reality  that   science  discovers are grounded   in   universals,  <br>which   are  common   features  and  characteristics of   the   entities  in reality  in  virtue  of  which   they  are  grouped <br>  into   circles  of  similars;  compare  Nominalism, Representationalism.                                                                                                                                                                                                |
| Ontology.                                        |                   |              |      |                                                                                                                                                                                                                                                                       | A representational artifact,  comprising  a taxonomy  as proper  part,  whose representations are intended to designate some <br>combination of universals, defined classes, and certain relations between them; see also Ontology, philosophical.                                                                                                                                                                                                                                                                                                         |
| Ontology, application.                           |                   |              |      |                                                                                                                                                                                                                                                                       | An application ontology  is an ontology  created to accomplish some task of local significance. See Ontology, reference.                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| Ontology, domain.                                | 领域本体          |              |      |                                                                                                                                                                                                                                                                       | A domain ontology is an ontology that describes and categorizes some domain.                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| Ontology, formal.                                | 形式本体          |              |      |                                                                                                                                                                                                                                                                       | The study of the universals, relations, and structures common to all domains of reality; also used to refer to an upper-level ontology such as BFO.                                                                                                                                                                                                                                                                                                                                                                                                        |
| Ontology, material.                              |                   |              |      |                                                                                                                                                                                                                                                                       | The study of the universals, relations, and structures common to some specific domain of reality; sometimes used as a synonym of “domain ontology.”                                                                                                                                                                                                                                                                                                                                                                                                        |
| Ontology, philosophical.                         | 哲学本体          |              |      |                                                                                                                                                                                                                                                                       | The theory of what exists—the study of the kinds of entities in reality and of the relationships that these entities bear to one <br>another; also known as metaphysics. The goal of philosophical ontology  is to provide a clear, coherent, and rigorously <br>worked out account of the basic structures of reality.                                                                                                                                                                                                                                    |
| Ontology, reference.                             | 参考本体          |              |      |                                                                                                                                                                                                                                                                       | An ontology  that is intended to provide a comprehensive representation of the entities in a given domain encapsulating the <br>terminological content of established knowledge of the sort that is contained in a scientific textbook.                                                                                                                                                                                                                                                                                                                    |
| Ontology, top- (or upper-) level.                | 上层本体          |              |      |                                                                                                                                                                                                                                                                       | An ontology  of highly general categories and relations that subsume the universals represented by specific domain ontologies.                                                                                                                                                                                                                                                                                                                                                                                                                             |
| Open-world assumption.                           | 开放世界假设      |              |      |                                                                                                                                                                                                                                                                       | The assumption that we capture knowledge within an ontology or ontology-like resource in an ongoing process as we discover it,<br> so that we can at no stage guarantee that we have discovered complete information—hence no conclusions should be <br>drawn from the fact that a given assertion is not recorded in our system; see General principles  of ontology design.                                                                                                                                                                              |
| Participation.                                   |                   |              |      |                                                                                                                                                                                                                                                                       | The relation between a material  entity and a process that obtains in virtue of the fact that the former participates in the latter.                                                                                                                                                                                                                                                                                                                                                                                                                       |
| Particular.                                      | 殊相              |              |      |                                                                                                                                                                                                                                                                       | An individual (nonrepeatable) denizen of reality (an instance of a universal); all particulars stand in the relation of instantiates to <br>some universal; each particular occupies a unique spatiotemporal location.                                                                                                                                                                                                                                                                                                                                     |
| Particular-particular relation.                  |                   |              |      |                                                                                                                                                                                                                                                                       | A relation between one particular and another; also called an instance-level relation; for example: Mary’s leg part_of  Mary; see Relation.                                                                                                                                                                                                                                                                                                                                                                                                                |
| Perspectivalism.                                 |                   |              |      |                                                                                                                                                                                                                                                                       | The view according to which reality is too complex and variegated to be embraced within  a single scientific theory, and that <br>multiple distinct scientific theories may be equally accurate representations of one and the same reality, for instance<br> because they partition this reality at different levels of granularity; see General principles  of ontology design.                                                                                                                                                                          |
| Principle  of reuse.                             | 复用原则          |              |      |                                                                                                                                                                                                                                                                       | A principle stating that ontologies should as far as possible reuse the ontological  content that has already been created;<br> see General principles  of ontology design.                                                                                                                                                                                                                                                                                                                                                                                |
| Process.                                         | 过程              |              |      |                                                                                                                                                                                                                                                                       | An occurrent  entity that exists in time by occurring or happening, has temporal parts, and always depends on at least <br>one independent continuant as participant.                                                                                                                                                                                                                                                                                                                                                                                      |
| Process boundary.                                | 过程边界          |              |      |                                                                                                                                                                                                                                                                       | An occurrent  entity that is the instantaneous temporal boundary of a process.                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| Quality.                                         | 量                |              |      |                                                                                                                                                                                                                                                                       | A specifically dependent continuant that, if it inheres in an entity at all, is fully exhibited, manifested, or realized in that entity.<br> In order for a quality to exist, one or more independent continuants must also exist. Examples include the mass of a kidney,<br> the color of this portion of blood, and the shape of a hand.                                                                                                                                                                                                                 |
| Realism.                                         |                   |              |      |                                                                                                                                                                                                                                                                       | The view that thought, experience, and knowledge are (if partially and fallibly) about reality. A view of this sort should be the <br>general attitude to be kept in mind throughout  the process of designing an ontology; see General principles  of ontology design.                                                                                                                                                                                                                                                                                    |
| Realizable entity.                               |                   |              |      |                                                                                                                                                                                                                                                                       | A specifically  dependent continuant entity that has at least one independent continuant as its bearer, and whose instances<br> can be realized (manifested, actualized, executed) in associated processes of specific correlated types in which the bearer participates.                                                                                                                                                                                                                                                                                  |
| Reflexivity.                                     |                   |              |      |                                                                                                                                                                                                                                                                       | The property of a relation R whereby anything that bears R to something also bears that relation to itself. The relation “is as tall<br> as” is reflexive, because when John is as tall as Jill, then he also stands in this same relation to himself.                                                                                                                                                                                                                                                                                                     |
| Relation.                                        |                   |              |      |                                                                                                                                                                                                                                                                       | The manner in which two or more entities are associated or connected together. BFO recognizes three basic types of relation:<br> connecting universal to universal, universal to particular, and particular to particular.                                                                                                                                                                                                                                                                                                                                 |
| Relational quality.                              |                   |              |      |                                                                                                                                                                                                                                                                       | A quality  that inheres in two or more independent continuants. Examples include a marriage bond, a debt, an agreement. <br>From the BFO perspective  there is both the relational quality universal marriage bond as well as specific instantiations<br> of this universal obtaining between (and specifically depending upon) John and Mary, Bill and Sally, and so forth.                                                                                                                                                                               |
| Representation.                                  |                   |              |      |                                                                                                                                                                                                                                                                       | An entity that makes reference to or is about another entity or entities.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| Representational  artifacts.                     |                   |              |      |                                                                                                                                                                                                                                                                       | A representation that has been produced by someone and made available in a form that allows it to be accessed by<br> others (such as a drawing, map, book, or computer database).                                                                                                                                                                                                                                                                                                                                                                          |
| Representationalism.                             |                   |              |      |                                                                                                                                                                                                                                                                       | The view that our perceptions, thoughts, beliefs, or models are directly about concepts (or ideas or images) in our minds and <br>only indirectly about nonmental entities of various sorts in reality. On this view, what we actually know are not things <br>in reality, but the ways in which we experience and conceptualize these things; see Idealism, Realism.                                                                                                                                                                                      |
| Representationalist interpretation of knowledge. |                   |              |      |                                                                                                                                                                                                                                                                       | The view that the goal of knowledge representation is to represent concepts or ideas; see Representationalism,  Idealism,<br> Realism.                                                                                                                                                                                                                                                                                                                                                                                                                     |
| Role.                                            |                   |              |      |                                                                                                                                                                                                                                                                       | A realizable entity that (1) exists because the bearer is in some special physical, social, or institutional set of circumstances in <br>which the bearer does not have to be, and (2) is not such that, if this realizable entity ceases to exist, then the physical <br>make-up of the bearer is thereby changed. A role is thus always optional.                                                                                                                                                                                                        |
| Semantic interoperability.                       | 语义互操作性      |              |      |                                                                                                                                                                                                                                                                       | A property that obtains between two (or more) data or information systems when they are such that, because their terms are <br>defined according to a common,  logically well-defined ontology,  each can carry out the tasks for which it was designed <br>using data and information taken from the other as seamlessly as it can when using its own data and information.                                                                                                                                                                               |
| Semantic Web.                                    | 语义网            |              |      |                                                                                                                                                                                                                                                                       | Whereas the World Wide Web (WWW) is an interconnected system of web pages, the Semantic Web (SW) is an interconnected <br>system made up of the content (data and information)  from those pages. The SW emerged out of thinking in the field of <br>artificial intelligence and was conceived as a system that enables machines to “understand” and respond to<br> complex human queries based on their meaning— hence the use of the term “semantic.”                                                                                                    |
| Site.                                            |                   |              |      |                                                                                                                                                                                                                                                                       | An immaterial entity in which objects such as molecules  of air or organisms can be contained.                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| Spatial region.                                  | 空间区域          |              |      |                                                                                                                                                                                                                                                                       | A continuant entity that  is a part of space. When an object moves from  one place to  another, it  occupies  a  continuous <br>  series  of   different   three-dimensional   spatial  regions  at   different times.                                                                                                                                                                                                                                                                                                                                     |
| Spatiotemporal region.                           | 时空区域          |              |      |                                                                                                                                                                                                                                                                       | An occurrent  entity at or in which occurrent  entities can be located. Just as the continuant representation of entities views<br> space as a container within  which objects and their qualities exist, so too the occurrent representation of processes views the <br>combination  of space and time together as such a container, within which processes unfold.                                                                                                                                                                                       |
| Specifically dependent continuant.               |                   |              |      |                                                                                                                                                                                                                                                                       | A continuant entity  that  depends on precisely one independent  continuant  for its existence. The former is dependent on the <br>latter in the sense that, if the latter ceases to exist, then the former will as a matter  of necessity cease to  exist also.<br> See Independent continuant, Generically  dependent continuant.                                                                                                                                                                                                                        |
| Symmetry.                                        | 对称性            |              |      |                                                                                                                                                                                                                                                                       | The property of a relation R whereby if a thing A bears  R to something else B, then B also bears R to A. Example:<br> if A is adjacent to B, then B is also adjacent to A.                                                                                                                                                                                                                                                                                                                                                                                |

| Temporal  region.                                | 时间区域          |              |      |                                                                                                                                                                                                                                                                       | An occurrent  entity that is a part of time.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| Term.                                            | 术语              |              |      |                                                                                                                                                                                                                                                                       | A noun or noun phrase, understood as a linguistic sign, that is utilized to represent some entity in the world.                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| Terminology.                                     | 术语集            |              |      |                                                                                                                                                                                                                                                                       | A representational artifact  containing a list of terms, complete with definitions, used in some domain and formulated in <br>a natural language.                                                                                                                                                                                                                                                                                                                                                                                                          |
